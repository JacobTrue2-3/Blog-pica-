# Проект на Django

## Установка зависимостей
```bash
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
```
## Приминиение миграций
```bash
python manage.py makemigrations (не обязательно)
python manage.py migrate
```
## Запуск сервера
```bash
python manage.py runserver
```
